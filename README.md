
# Windows Driver pack for Samsung platforms

This repository contains driver binary files for Samsung.
All driver binary files form a board support package to be used on Samsung devices to provide hardware support for the Windows operating system.

These driver files are not perfect, typos may exist, feel free to file an issue on GitHub in case you found any.
